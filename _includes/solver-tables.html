Key:
  • Linear/Quadratic = Supports Linear Programming possibly with the addition of quadratic objectives or constraints.
  • Convex conic = Supports Second-order Cone Programming or Semidefinite Programming or exponential cones.
  • Convex smooth = Supports constrained derivative-based Nonlinear Programming with convex constraints and objective functions
  • Nonconvex = Supports general nonconvex nonlinear programming.
  • Integer = Supports integer-constrained variables, e.g., Integer Programming.
  • cb = Supports Integer Programming callbacks.
